SushiCode

> Sushis elegantes e saborosos.
> Minimalismo japonês com identidade visual inspirada no universo tech.

---

Sobre o projeto

O **SushiCode** é uma landing page interativa para um restaurante de sushi voltado ao público tech.

O projeto une:

* design clean
* experiência moderna
* microinterações
* carrinho funcional
* checkout interativo
* temática inspirada em programação

---

Funcionalidades

Carrinho inteligente

* adicionar produtos
* remover itens
* alterar quantidade
* cálculo automático
* contador dinâmico

Checkout

* entrega ou retirada
* formulário interativo
* cálculo total
* confirmação de pedido

Interface

* layout responsivo
* animações suaves
* drawer lateral
* modal de checkout
* identidade visual minimalista
